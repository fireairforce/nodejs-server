## some git types
- feat :新功能 
- fix :修复bug  
- doc : 文档改变
- style : 代码格式改变
- refactor :某个已有功能重构
- perf :性能优化
- test :增加测试
- build :改变了build工具 如 grunt换成了 npm
- revert: 撤销上一次的 commit 

## download
```
npm install -g anydoor
```

## useage
```
anydoor        # start the server

anydoor -p 8080 # set the port as 8080

anydoor -h localhost #set the port as localhost
```
